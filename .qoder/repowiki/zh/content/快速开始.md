## 简介

cut-video是一个基于阿里云百炼FunASR API的智能视频剪辑工具。它提供了强大的语音识别能力，支持词级时间戳输出、热词增强、视频文件自动音频提取等功能。该工具既可以通过命令行使用，也可以通过Web界面进行交互式操作。

主要特性：
- 长音频转写（最长12小时，2GB文件）
- 支持中文、英文等多语言（含方言）
- 词级时间戳输出（毫秒级精度）
- 语气词过滤
- 视频文件支持（自动用ffmpeg提取音频）
- 热词支持（v1和v2模型）
- 命令行工具 + Python API
- Web界面视频剪辑（词级时间戳可视化编辑）

1. 登录 [阿里云百炼控制台](https://bailian.console.aliyun.com/)

在项目根目录创建`.env`文件或设置环境变量：


确保系统已安装以下必需组件：

验证安装：

## 基础使用示例

### 1. 命令行基本用法

#### 基础音频转写
```bash
# 基本转写（默认paraformer-v1 + 热词）
python cli.py audio.wav -o result.txt

# 输出带时间戳的结果
python cli.py audio.wav -t -o result.txt

# 使用v2模型
python cli.py audio.wav -m paraformer-v2 -t -o result.txt
```

#### 视频文件处理
```bash
# 直接转写视频（自动提取音频）
python cli.py video.mp4 -t -o result.txt

# 指定视频提取音频的保存位置
python cli.py video.mp4 --audio-output audio.wav -o result.txt
```

### 2. Python API使用

#### 基本转写
```python
from src.transcriber import FunASRTranscriber, ModelType

transcriber = FunASRTranscriber()

# 基本转写（v2模型）
result = transcriber.transcribe(
    audio_file_path="audio.wav",
    model=ModelType.PARAFORMER_V2,
)
print(result.text)
```

#### 热词增强转写
```python
# v2模型热词
result = transcriber.transcribe(
    audio_file_path="audio.wav",
    model=ModelType.PARAFORMER_V2,
    vocabulary_id="your_vocabulary_id",
)

# v1模型热词
result = transcriber.transcribe(
    audio_file_path="audio.wav",
    model=ModelType.PARAFORMER_V1,
    phrase_id="your_phrase_id",
)
```

## 高级功能演示

### 1. 热词功能

#### 热词配置文件
项目自带`hotwords.json`配置文件，包含预设的热词：

```json
{
  "运维": 5,
  "网安": 5,
  "HTTP": 2,
  "AI": 2,
  "Paraformer": 2,
  "赛博永生": 4
}
```

#### 热词限制
- 最多500个热词
- 中文热词≤10字符
- 英文/混合热词≤5词
- 权重范围：[1,5]增强，[-6,-1]减弱

### 2. Web界面视频剪辑

#### 启动Web服务
```bash
# 进入Web目录
cd cut-video-web

# 安装依赖
uv sync

# 设置API Key并启动服务
export DASHSCOPE_API_KEY='your-api-key'
uvicorn backend.main:app --reload --port 8000
```

#### 使用流程
1. 打开 `http://localhost:8000`
2. 上传视频文件（拖拽或点击选择）
3. 等待ASR转写完成（自动使用v1+热词）
4. 点击词标记删除（红色删除线）
5. 勾选"烧录字幕"可选项
6. 点击"执行剪辑"
7. 下载剪辑后的视频

## 最佳实践

### 1. 模型选择策略

| 场景 | 推荐模型 | 说明 |
|------|----------|------|
| 中文访谈、讲座 | paraformer-v1 | 中文效果好，热词支持更好 |
| 多语种会议 | paraformer-v2 | 多语种支持更好 |
| 通用场景 | 默认paraformer-v1 | 性价比高 |

### 2. 热词配置优化

- **权重设置**：常用词使用较高权重（3-5），生僻词使用较低权重（1-2）
- **热词粒度**：避免过于具体的短词，优先使用有意义的词汇组合
- **定期更新**：根据实际使用情况调整热词权重

### 3. 性能优化建议

- **批量处理**：大量文件时使用后台任务队列
- **内存管理**：大文件转写时注意内存使用
- **网络稳定性**：确保稳定的网络连接以避免转写中断

### 4. 错误处理策略

- **文件格式**：确保输入文件格式受支持
- **权限设置**：确保读写权限正确
- **磁盘空间**：预留足够的磁盘空间用于中间文件

## 故障排除

### 1. 环境变量问题

**问题**：API Key未设置
```bash
# 错误信息
错误: 请设置 DASHSCOPE_API_KEY 环境变量
export DASHSCOPE_API_KEY='your-api-key'
```

**解决方案**：
```bash
# 检查环境变量
echo $DASHSCOPE_API_KEY

# 重新设置
export DASHSCOPE_API_KEY='your-correct-api-key'
```

### 2. ffmpeg相关错误

**问题**：ffmpeg命令不可用
```bash
# 错误信息
音频提取失败: ffmpeg: command not found
```

**解决方案**：
```bash
# macOS安装ffmpeg
brew install ffmpeg

# Ubuntu/Debian安装ffmpeg
sudo apt update
sudo apt install ffmpeg

# Windows安装ffmpeg
# 从官网下载并添加到PATH
```

### 3. 依赖安装问题

**问题**：uv安装失败
```bash
# 解决方案：使用pip作为后备方案
pip install -e .
```

### 4. 热词创建失败

**问题**：热词创建API调用失败
```bash
# 检查热词格式
cat hotwords.json

# 确保权重在有效范围内
# 权重范围：[1,5]增强，[-6,-1]减弱
```

### 5. Web服务启动问题

**问题**：端口被占用
```bash
# 更换端口
uvicorn backend.main:app --reload --port 8001

# 查找占用端口的进程
lsof -i :8000
```

### 6. 文件权限问题

**问题**：无法读取/写入文件
```bash
# 检查文件权限
ls -la uploads/
ls -la outputs/

# 修复权限
chmod -R 755 uploads/
chmod -R 755 outputs/
```
